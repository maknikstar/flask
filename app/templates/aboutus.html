Наша команда школьников создала инновационную систему
распознавания лиц на базе Flask!

С помощью Python и библиотек компьютерного зрения мы разработали алгоритм, который определяет лица на загруженных изображениях. Веб-интерфейс на HTML и CSS позволяет легко загружать фото и мгновенно видеть результат. Этот проект стал отличной возможностью освоить веб-разработку, работу с API и основы машинного обучения.

Технологии: Flask, Python, OpenCV, HTML. Цель: Упростить анализ изображений через простой и понятный инструмент..
